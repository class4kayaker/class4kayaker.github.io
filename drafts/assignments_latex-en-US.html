Advice on Submitting Assignments in LaTeX

Mon 17 September 2018 By Jonathan Robey

Introduction

LaTeX is a common format/language for typesetting documents. It is one of the most common formats accepted for article submissions by journals in the field of mathematics.

As such, using it for the write-ups for class assignments both provides practice and the PDF documents produced are reasonable for submission.

A useful reference for many questions is the Wikibook.

Installation

LaTeX

If you are using Ubuntu or a Ubuntu derived distribution, a reasonable package to install would be texlive-latex-recommended.

LaTeXrun

One of the major issues with LaTeX is that due to a legacy of its development, the program both requires that it be run multiple times to create the actual final document, and the errors it encounters are buried in a large stream of text making it difficult to identify them for correction.

latexrun is a python script that is intended to solve both of these issues. The script runs the correct pattern of pdflatex and bibtex until there is no change in the source files, isolates the set of files produced as intermediate products, and lists the errors in a much more readable fashion.

Structure

An example latex document is shown below

\documentclass{article}

% Use graphics package
\usepackage{graphicx}

% Use particular bibliography package
\usepackage{natbib}

% Add submission information
\title{Submission title}
\date{Todays Date}
\author{Jonathan Robey}

\begin{document}

    % Typeset title
    \maketitle

    % Section heading
    \section{Question 1}

    Section text here.

    \subsection{Convergence}

    Subsection text

    % Figure with caption
    \begin{figure}

        \includegraphics[width=\textwidth]{FIGURES/convergence_graph}

        \caption{A graph of the convergence rate for Euler's method}
        \label{fig:convergence_graph}
    \end{figure}

    % Table of data
    \begin{table}
        \begin{tabular}{|l|r|r|}
        $h$ & $E$ & Rate \\
        \hline
        $2^{-3}$ & $3.14\cdot10^{-2}$ & \\
        \vdots & & \\
        \end{tablular}
    \end{table}

    % Add bibliography
    \section*{References}

    \bibliographystyle{plain}

    \bibliography{TEXT}

\end{document}

Useful Packages

As LaTeX is extensible, there are a number of useful non-standard packages, which are included in the Ubuntu texlive-latex-recommended package.

Graphicx

A standard package to improve inclusion of graphics into a LaTeX document.

Natbib

One of the more modern bibliography typesetting packages

PGFPLOTSTABLE

In cases such as convergence studies, it is necessary to include a table of calculated data into the document. This can be done manually, but this adds greater potential for including an error when typing in the data, or have difficulties when adding new datapoints. Alternatively, PGFPLOTSTABLE allows the extraction of data from structured CSV files into nicely formatted tables. While this does require that the originating data be arranged correctly this can easily be done in an automated fashion and the ability to both separate out the data into a separate file helps avoid uncertainties in which dataset is currently included.

Proudly powered by Pelican, which takes great advantage of Python.
